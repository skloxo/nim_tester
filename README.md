# API 模型测试平台 (API Model Benchmarking Platform)

这是一个基于 **TypeScript + Bun** 构建的高性能 API 模型评测框架，专为评估 NVIDIA NIM 及 OpenAI 兼容接口的大模型（LLM、VLM、Embedding、Reranker、TTS 等）设计。

平台支持并发测速、用例测试、多维自动评分、场景智能推荐以及 Excel 报告一键导出。

---

## 🚀 快速开始 (Windows + WSL 混合环境一键启动)

本项目的启动入口已针对 Windows 双击运行进行了极致优化，支持 **本地 WSL 直接启动** 和 **Docker 容器化一键部署**。

### 方式一：Docker 模式 (推荐，开箱即用)
适用于本地拥有 Docker (运行在 WSL 内部) 的环境，服务完全运行在独立的容器中，不污染本地环境。
1. 在 Windows 资源管理器中双击运行项目根目录下的 **`启动Docker.bat`**。
2. 脚本会自动在 WSL Docker 容器内构建并启动服务，且在 **0 延迟** 下自动为您在浏览器中打开主控制台：[http://localhost:28080](http://localhost:28080)。
3. **关闭服务**：如需停止容器，请在项目根目录下执行命令：`wsl docker compose down`。

### 方式二：本地模式 (前台黑框/支持开发调试)
适用于希望直接运行代码、调试或修改评测逻辑的场景。
1. 双击运行项目根目录下的 **`启动WebUI.bat`**。
2. 脚本会自动检测本地依赖 `node_modules`。如果缺失，会自动在 WSL 内运行安装服务。
3. 浏览器会自动拉起跳转至控制台。命令行黑框前台保持运行并实时刷新测试日志。
4. **关闭服务**：直接关闭该 CMD 黑框窗口即可彻底停止服务，无残留后台进程。

## ⚙️ 配置与 API 密钥

在开始运行大模型测试前，您需要在项目中配置您的 NVIDIA API 密钥或其他模型供应商的密钥。

### 1. 配置文件模式 (推荐)
直接修改根目录下的 **`config.yaml`**：
* 找到 `api_keys:` 节点，将占位符替换为您真实的 API 密钥（每行一个，支持多 Key 自动滑动窗口轮换和自动限速避让）：
  ```yaml
  api_keys:
    - "nvapi-YOUR_REAL_KEY_1"
    - "nvapi-YOUR_REAL_KEY_2"
  ```
* 可以在 `testing:` 节点下修改全局并发度（`concurrency`）和速率限制等参数。

### 2. 网页端直接配置
打开控制台网页 [http://localhost:28080](http://localhost:28080) 后，在最上方的 **“API 档案”** 板块中，可直接在文本域里粘贴多个 API 密钥，点击 **“保存当前”** 即可存入数据库，下次测试时一键切换。

## 📊 评测维度与报告

平台目前对模型进行 9 大核心类别的智能化测试，包含：
* **通用对话** (`general_chat`)
* **视觉语言 VLM** (`vision_language`)
* **推理/思维链** (`reasoning`)
* **代码** (`code`)
* **文本嵌入** (`text_embedding`)
* **图文嵌入** (`multimodal_embedding`)
* **重排序** (`reranker`)
* **图像生成** (`image_generation`)
* **语音音频** (`audio`)

### 多维评估规则：
* **硬性缺陷限制**：若模型最大上下文 `< 32k`，则自动判定其限制使用场景，并扣减 15 分，Grade 等级最高限制为 B 级；若模型 `avg_tps < 3.0`，则判定极慢响应，扣减 20 分，Grade 等级最高限制为 C 级。
* **智能场景推荐**：评测结束后，系统会根据模型的参数规模、实测 TPS 吞吐、最大上下文等参数综合判定其适合的应用场景，给出如“AI Agent 开发”、“高频实时交互”、“单意图路由”等推荐标签。
* **Excel 报告导出**：在网页端点击 **“导出 Excel”** 可瞬间下载多 Sheet 详细评测报告（包含总体分数排序、具体测试用例执行耗时、TPS、及推荐场景）。
